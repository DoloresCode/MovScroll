# MovScroll

## Description
 
MovScroll is a user-friendly web application dedicated to all movie enthusiasts and cinephiles. With an extensive movie and actor database, it allows users to easily navigate, search, and manage their favorite films and actors.

Key Features:

- **Actor Index:** Users can browse through an extensive list of actors, view their detailed profiles, and even add new actors to the list.
- **Movie Management:** Users have the ability to view a list of movies, add new movies, and edit or delete existing ones. They can also associate movies with actors in the database.
- **Watchlist:** For better organization and tracking, users can create watchlists where they can add their favorite or yet-to-watch movies. Each watchlist can be updated and deleted as needed.

## User Flow

1. As a user, I should be able to access the app and see the home and about pages.
2. As a user, I should be able to access the list of actors and movies.
3. As a user, I should be able to view details about each actor, including the movies they're associated with.
4. As a user, I should be able to add new actors to the actor list.
5. As a user, I should be able to update and delete an actor on the actor list.
6. As a user, I should be able to add new movies to an actor's filmography.
7. As a user, I should be able to view, add, update, and delete movies independently, without being associated with an actor.
8. As a user, I should be able to view the list of watchlists and the detailed view of each watchlist.
9. As a user, I should be able to create new watchlists.
10. As a user, I should be able to add movies to a watchlist.
11. As a user, I should be able to update and delete a watchlist.

## Problem MovScroll Solves

MovScroll is a user-friendly platform designed to enhance the movie-viewing experience by providing a centralized, searchable database for movie and actor information, as well as personalized watchlists. By offering tools for movie tracking, discovery, and personal curation, it simplifies movie management and deepens the engagement with cinema.


## Who would use it

MovScroll can be used by anyone with a love for cinema (what ever your age), from casual movie-watchers who need an organized way to keep track of films they want to see, to serious cinephiles who want detailed information about movies and actors at their fingertips. It's also a useful tool for film critics, movie bloggers, and students of film and media studies who require an easy-to-use, comprehensive database for their work.
